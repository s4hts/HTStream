# Release Process

## Github release
- create a release branch `release-1.4.0`
- Update the version here:  https://github.com/s4hts/HTStream/blob/master/common/src/version.h.release e.g. `1.4.0`
- commit the change
- merge to master
- tag that commit `git tag 1.4.0`
- https://github.com/s4hts/HTStream/releases/new create a new release here
